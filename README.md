# Requirements

- Install required Python modules with `pip install -r requirements.txt`
- Create .env file with following variables
    - `EMAIL` being your Confluence account email address.
    - `TOKEN` being your [Confluence API token](https://support.atlassian.com/atlassian-account/docs/manage-api-tokens-for-your-atlassian-account/)
    - `URL` being Confluence root URL.
